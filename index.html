Stable Diffusion API Usage

Use the following API endpoint to generate images from text prompts:

POST /generate

Example Request

    curl -X POST "http://localhost:8000/generate" \
    -H "Content-Type: application/json" \
    -d '{"prompt": "Astronaut in a jungle, cold color palette, muted colors, detailed, 8k"}'

The response will contain a base64-encoded image that can be displayed in your project.
